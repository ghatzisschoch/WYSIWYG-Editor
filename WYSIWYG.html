User Interface documentation:

Buttons:

Make New Element: Creates a new element of any time (e.g. button, div, textarea, input.

Change Color of Selection: Prompts and changes the color of the current selection.

Delete Current Selection: Removes the current selection from the page (Shortcut: DEL key).

Edit Selection onClick(): Prompts and changes the onclick function of the current selection. JavaScript functions which are added using the \" Add JavaScript functions \" button are able to be called here.

Add JavaScript functions: Displays a textarea in which JavaScript code and functions can be input. The code will be added to the generated page in a SCRIPT element.

View Selection Code: Displays all of the user-generated elements on the page.

Search & Select elements by ID: Prompts for the id of the desired element, and if found displays the element's code and selects the element.

Hide page code: Hides the user-generated code textarea. Click again to show the page code

Copy page code: Alerts a model box which is copyable through the use of CRTL+C.

Save page: Saves all user-generated elements using localStorage. This allows the page to be saved for later use.

Load page: Loads all user-generated elements saved previously using localStorage. This will NOT delete any current elements from the page, and will only append saved elements to the page.

Other UI Functions:

The UI "toolbox" and page code-area can be dragged around the page by simply right-clicking on it and moving the mouse. Releasing the right mouse-button will stop the dragging.

The UI can be hidden using the "Hide UI" button, which is always found in the top right corner of the page.

UNDO CHANGES: To undo changes press CTRL+Z.
